# 디자인 파일

이 폴더는 공학용 전자계산기 웹앱의 UI/UX 디자인 참조 파일을 포함합니다.

## 파일 목록

### 1. `code.html`
- **설명**: 완성된 UI 구조 및 스타일 코드
- **기술**: HTML + Tailwind CSS
- **용도**: 
  - UI 컴포넌트 구조 참조
  - 스타일링 가이드
  - 레이아웃 및 반응형 디자인 참조

### 2. `screen.png`
- **설명**: 계산기 UI 디자인 스크린샷
- **용도**:
  - 최종 디자인 비전 확인
  - 색상 및 간격 참조
  - 다크 모드 디자인 확인

## 디자인 시스템

### 컬러 팔레트

**다크 모드** (기본):
- Primary: `#2b6cee`
- Background: `#101622`
- Surface: `#151c2b`
- Text: `#ffffff`

**라이트 모드**:
- Primary: `#2b6cee`
- Background: `#f6f6f8`
- Surface: `#ffffff`
- Text: `#0f172a`

### 타이포그래피
- Display Font: Space Grotesk
- Body Font: Noto Sans

### 아이콘
- Material Symbols Outlined

## 참조 방법

구현 시 이 디자인 파일들을 참조하여:
1. UI 컴포넌트 구조 설계
2. 스타일 시스템 구축
3. 반응형 레이아웃 구현
4. 다크/라이트 모드 테마 적용
